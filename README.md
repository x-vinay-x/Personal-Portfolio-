# Vinay N - Portfolio Website

A modern and responsive portfolio website showcasing my work as an Embedded Systems Engineer with AI/ML integration expertise. Built with Next.js and styled using Tailwind CSS, featuring projects in Verilog HDL, Embedded C, IoT systems, and AI/ML applications.

## About

Embedded Systems Engineer specializing in Verilog HDL, Embedded C, and hardware-software co-design. Currently working as a Prompt Engineer on HIPAA-compliant healthcare AI chatbots, with experience in drowsiness detection systems, IoT monitoring, and FPGA-based traffic controllers.

## Features

- **Responsive Design**: Optimized for all devices, including mobile, tablet, and desktop.
- **Dark Theme**: Provides a visually appealing dark theme.
- **Smooth Animations**: Utilizes Framer Motion for animations.
- **Project Showcase**: Features embedded systems, IoT, and AI/ML projects.
- **Certifications Display**: Showcases professional certifications and training.

## Technologies Used

- **Next.js**: React framework for building static and dynamic websites.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Framer Motion**: Animation library for React to create smooth transitions and effects.
- **GitHub API**: Fetches the latest GitHub projects dynamically.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/x-vinay-x/portfolio-website.git
   ```

2. Navigate into the project directory:

   ```bash
   cd portfolio-website
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
